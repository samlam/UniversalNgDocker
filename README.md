# README

## local dev machine

```
npm install -g npm@latest @angular/cli nodemon@1.17.0 browser-sync
```

## debug start

```
cd ./web

npm install

npm run build:dev
```

## running the container

```
docker-compose up
```

## web browser

navigate to `localhost:8000`

browser-sync ui is running on `http://localhost:3001/`
